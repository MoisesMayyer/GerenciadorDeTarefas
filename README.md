# Gerenciador de Tarefas

Projeto desenvolvido em Python para gerenciamento de tarefas através de uma interface de linha de comando (CLI).

## Funcionalidades

- Criar tarefas
- Editar tarefas
- Excluir tarefas
- Listar tarefas
- Alterar status das tarefas
- Persistência de dados em JSON

## Tecnologias utilizadas

- Python 3
- JSON
- Programação Orientada a Objetos (POO)

## Objetivo

Projeto criado para praticar:

- CRUD
- Manipulação de arquivos
- Modularização
- Tratamento de exceções
- Conceitos básicos de POO
